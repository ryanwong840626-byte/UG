# Project Context

## Project name

UG / NX workflow research and development, focused on lens mold automation

## Objective

Build a stable workspace for researching, documenting, and implementing UG / NX related workflows, with the current focus on lens mold surface modeling and CAM automation.

## Background

The repository is intended to preserve project memory so setup steps, technical findings, scripts, and decisions do not get lost between work sessions. The current production goal is non-spherical and freeform lens mold modeling followed by tool path generation in NX.

## Current goals

- automate lens mold geometry generation in NX
- build a parameter-driven path from optical requirements to mold geometry
- prepare CAM automation for machining verification and test-cut iteration

## Expected deliverables

- documented setup process
- reusable development notes
- lens mold parameter templates
- NX automation scripts for geometry and manufacturing preparation
- versioned project history in GitHub

## Key tools

- Siemens NX / UG
- Git
- GitHub
- Codex
- PowerShell

## Working assumptions

- the repository will be used for both documentation and code
- early work will focus on workflow validation before large-scale development
- project details will continue to evolve as lens and mold requirements become clearer
- the first implementation path will prioritize rotational asphere geometry before freeform surfaces
